## Social Spark Suggestions
### General info 
This app is designed to make social interactions easier in the midst of busy work and school schedules. The user inputs the time of day, for example; "Morning," "Mid-morning," "Afternoon," "Afternoon Snack-Time," "After Dinner"
The user must input the time of day as is written in the example above. The textfield is case sensitive.
For each time of day, the user will receive a suggestion on what social activity they must participate in. 
Each suggestion, referred to in the app by the term "social spark suggestion" is helpful in making social interactions more efficient, meaningful and strengthens the user's social skills and bonds with their loved ones. 
If the user incorrectly iputs the time of day, a helpful error message will be displayed that will gently point the user in the right direction.

### Design Considerations
The app is designed to be fun, engaging and intuitive to the user. The use of bright and colourful imagery and interesting fonts aids in this. In addition to this, there is a welcome screen that sets the tone for the main section of the app, this enhances the level of engagement users will feel with the app.
Ease of use is also a priority, and that is why the User Interface is designed with simplicity, while maintaining visual appeal. 
#### Github and Github Actions
The use of Github and Github Actions has aided in the development of the app. 
In particular, Github Actions has ensured that every addition of code to the app runs and functions as intended. 
With the use of automated build tests of each version of the app, the app has been efficiently tested with every addition of code.
In the event of errors having occured, Github Actions allowed the app to be reverted to a state prior to an error ocurring, this made troubleshooting and debugging much easier and more efficient. 
Pushing app files to Github also made for a more secure and safe means of storing information and data. 
It also allowed for the app to be worked on from multiple devices, by use of the Github Desktop application.
